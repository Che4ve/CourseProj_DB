# План улучшения дизайна фронтенда

## Цели

Улучшить визуальный дизайн интерфейса, сделав его более современным, понятным и приятным для использования. Основной фокус на карточках привычек, навигации и модальных окнах.

- `Tabs`, `TabsList`, `TabsTrigger`, `TabsContent`
- Поддержку активного состояния
- Адаптивный дизайн

## Технические детали

### Иконки

Использовать библиотеку `lucide-react` (уже используется в проекте):

- `CheckCircle2` или `TrendingUp` для хороших привычек
- `XCircle` или `TrendingDown` для плохих привычек
- `Plus` для кнопок добавления
- `User` для кнопки профиля
- `Calendar`, `Clock`, `Bell` для модалок

### Стилизация

- Использовать существующую систему дизайна (shadcn/ui, Tailwind CSS)
- Следовать принципам доступности (контрастность, размеры кликабельных элементов)
- Обеспечить адаптивность на мобильных устройствах

### Цветовая схема

- Хорошие привычки: зеленые оттенки (success colors)
- Плохие привычки: красные/оранжевые оттенки (destructive/warning colors)
- Использовать существующие CSS переменные из `globals.css`
